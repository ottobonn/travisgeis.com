Flashcards

4 June 2010

About a 1-minute read

I like to keep multiple projects open at the same time. It keeps me well-rounded.

Anyway, a while back I was looking for an open-source web solution to studying with traditional-style, basic black-and-white flash cards. My dad pointed out that main problem with the options online is in the way users create cards: they must enter each side of each card in its own box, a process totally incompatible with the thousands of words people already have listed in vocabulary and study-guide documents.

So, I thought it might be a fun idea to work up my own flashcard application, using plain text exportable from spreadsheet editors to upload the card decks. It may not catch on, but it’s worth a try.

Soon the flashcards project will have its own page. I already have a good amount of code written, and hopefully I’ll have a working sub-domain with this web app running soon.
